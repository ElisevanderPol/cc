Code Cult Programma:
Het Code Cult programma ziet er als volgt uit voor de 18 weken:
Les 1: Hour of Code
Les 2-4: Scratch
Les 5-9: Python
Les 10-18: Project
Iedere les duurt 1u40 waarbij een Code Cult docent aanwezig is om te ondersteunen en context te geven aan de opdrachten. Centraal staat de vrijheid van de leerlingen. Programmeren is een te breed onderwerp om een monoliet programma te bedenken. Niet alle leerlingen hebben dezelfde interesses en deze diversiteit is wat programmeren leuk maakt. Het idee is om de leerlingen in aanraking te brengen met hoe zij zelf een idee kunnen vormen en dit kunnen uitwerken. Belangrijk is dat de leerlingen leren hoe zij informatie op moeten zoeken, zodat zij ook na het lesprogramma in staat zijn om dit soort projecten zelf te starten. Naast de fysieke lessen bieden wij ook on-line ondersteuning in de vorm van een platform waarbij de leerlingen vragen kunnen stellen en problemen en achievements kunnen delen. Het idee is dat de leerlingen elkaar ook ondersteunen hierbij. Een belangrijk component van programmeren is het sociale element. Programmeurs kunnen veel van elkaar leren en het is vaak erg nuttig om een ander perspectief te krijgen op een probleem. 
Les 1: Hour of Code
In les 1 gaan wij aan de slag op code.org. Het is hierbij de bedoeling dat de leerlingen een gevoel krijgen bij formele oplossingen verzinnen bij problemen. Tevens worden zij blootgesteld aan de basis controle structuren in programmeren. Het idee is dat zij de eerste 3 niveaus halen (Doolhof en Artiest 1 en 2) Deze les is voornamelijk bedoeld als introductie en voorbereiding op de Scratch lessen.

Les 2-4: Scratch
De lessen met Scratch hebben als nut om de creatieve vaardigheden van de leerlingen te activeren. Het is hierbij belangrijk dat de leerlingen zich niet beperkt voelen door de opdrachten, maar ook niet verdwalen in de mogelijkheden. Om dit te ondersteunen zijn er 3 spellen die gemaakt kunnen worden aan de hand van componenten. Deze componenten laten zien hoe een spel is opgebouwd en waar je allemaal rekening mee moet houden tijdens het bouwen van een spel. De 3 spellen zijn pong, breakout en snake. Deze spellen worden vaak genoemd als beste spellen om mee te beginnen, maar als de leerlingen iets anders willen proberen dan zal Code Cult dit proberen te ondersteunen.

Les 5-9: Python
Tijdens de Scratch lessen komen de leerlingen in aanraking met de mogelijkheden van programmeren. Met Python gaat er weer een nieuw gebied open. De leerlingen gaan hierbij de basis leren achter echt programmeren. Tijdens deze lessen gaan zij zelf tekenen naar het scherm en de objecten op het scherm aansturen. In de lessen leren zij over data-structuren en hoe ze deze kunnen manipuleren. De bedoeling is om aan het eind een visualisatie te hebben van een onderwerp dat de leerlingen zelf kunnen kiezen.

Les 10-18: Project
Bij het project mogen de leerlingen zelf kiezen wat zij gaan bouwen. In de eerste les gaan wij met de leerlingen bespreken wat zij zouden willen maken en kiezen wij samen met hun de juiste technologie om dit te bereiken. De leerlingen zijn volledig vrij in het onderwerp. Of het nu gaat om een site, een spel, computer gegenereerde muziek, een gedichtenschrijver, een twitter-bot, een visualisatie, wij zullen alles proberen te ondersteunen en er voor zorgen dat het project haalbaar is in de 9 weken die hiervoor staan. Aan het eind is het de bedoeling dat de leerlingen iets hebben gemaakt waar zij trots op zijn.

Technologie
Hour of Code en Scratch zijn allebei on-line beschikbaar. Scratch kan ook geinstalleerd worden. Python kan grotendeels on-line geoefend worden, maar moet uiteindelijk ook geinstalleerd worden om makkelijker een groter programma te kunnen managen.

Tijdens de Code Cult weken, werken de leerlingen met Github. Github is een platform om code te delen waarbij je oudere versies terug kan vinden. Dit is handig omdat zij hier meteen een basis hebben voor een portfolio en omdat het inzichtelijk wordt wie wat gedaan heeft.

Ondersteunend Materiaal
Om de leerlingen te ondersteunen bieden wij on-line materiaal aan. Dit materiaal zal bestaan uit kleine oefeningen en uitleg over verschillende elementen van het programmeren. Het idee is dat de leerlingen grotendeels zelfstandig verder kunnen komen en gewend raken aan het zelf oplossen van problemen. Voor de verschillende onderdelen zal materiaal ontwikkeld worden om dit te faciliteren. Voor Scratch is er een online “studio” waar verschillende componenten van spellen uitgewerkt zijn. Dit wordt gecombineerd met les-materiaal om leerlingen te trainen in het uiteenzetten van de spel-componenten. Voor het Python gedeelte zal er een les-programma gemaakt worden waarbij de leerlingen leren de opgedane kennis uit Scratch over te zetten naar deze programmeertaal. Dit begint bij het begrijpen van functies en data structuren en bouwt langzaam uit naar programma flow en controle. Hierbij zal ook veel verwezen worden naar verschillende on-line bronnen om bepaalde kennis extra te kunnen oefenen.
Beoordeling
Omdat de leerlingen vrij zijn om zelf een pad te kiezen is het erg moeilijk om een traditioneel cijfer te geven. De bedoeling is dan ook om er voor te zorgen dat de leerlingen bereiken wat zij zelf willen bereiken tijdens deze weken. Om dit te ondersteunen, biedt Code Cult een systeem aan waarbij de verschillende elementen van programmeren inzichtelijk gemaakt worden per leerling. Sommige leerlingen zullen zich meer op sites richten, andere op games. Om dit soort diversiteit goed te beoordelen moet gekeken worden naar welke onderdelen de leerling op geoefend heeft. Hier is dus geen centrale toets voor nodig, maar in plaats daar van werken wij met een dynamisch badge systeem. Als een leerling iets voor elkaar heeft gekregen dan krijgt de leerling een badge voor hetgeen hij of zij heeft gedaan. Deze badges worden op een profiel weergegeven zodat het voor een docent direct duidelijk is hoeveel de leerling heeft gedaan en hoe goed hij hier in geworden is.
